## Para quien es TrackFish

TrackFish esta disenado para **frigorificos y plantas procesadoras de pescado** de cualquier tamano. Si tu operacion involucra:

- Recibir capturas o compras de pescado
- Procesar y empaquetar cajas para almacenamiento o venta
- Despachar producto a clientes con camiones propios
- Controlar la parte financiera (ventas, gastos, cuenta corriente)

TrackFish cubre todo tu flujo de trabajo en una sola aplicacion.

## Que podes hacer

## Tipo de aplicacion

TrackFish es una **aplicacion de escritorio para Windows** que se conecta al servidor central. Requiere una conexion a internet activa para operar.

<Info>
Necesitas conexion a internet para usar TrackFish. Todos los datos se almacenan de forma segura en el servidor — no se pierden si cambias de computadora o reinstales la app.
</Info>
